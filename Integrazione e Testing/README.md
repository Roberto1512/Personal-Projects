# Integrazione e Testing — The Verifiers

## Descrizione

Cartella dedicata alle attività di **collaudo, integrazione e validazione** del progetto "The Verifiers", sviluppato nell'ambito del corso di Integrazione e Testing del Software.

Questo modulo è strettamente collegato all'implementazione web Flask contenuta nella cartella `Ingegneria del Software`.

---

## Contenuto della Directory

### Documentazione

- **`DocumentazioneITSS - TheVerifiers.docx`** (~3.1 MB)
  Documento principale di *Integration and Testing Software Specification* (ITSS). Include:
  - Strategia di test (approccio, livelli di test e priorità).
  - Test Plan con i test case definiti.
  - Risultati attesi e report di esecuzione.
  - Matrici di tracciabilità requisiti ↔ test.

### Directory di Test

- **`Test/`**: Directory predisposta per ospitare codice automatizzato di test (es. test unitari, test di integrazione, test end-to-end). Attualmente vuota.
- **`TestHW2/`**: Directory predisposta per l'Homework 2 del corso (suite di testing avanzata). Attualmente vuota.
