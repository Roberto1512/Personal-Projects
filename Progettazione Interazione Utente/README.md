# Progettazione Interazione Utente (PIU)

## Descrizione

Documentazione prodotta per il corso di **Progettazione dell'Interazione Utente** (A.A. 2023-2024). Il progetto copre l'intero ciclo di *User-Centered Design* applicato a due sistemi: **The Verifiers** e **FitMate**.

Le fasi documentate comprendono: ricerca utente, analisi dell'utenza potenziale, creazione di Personas, task analysis, prototipazione a bassa e alta fedeltà, test di usabilità.

## Metodologia

Il progetto segue il framework di *User-Centered Design* (UCD):

1. **Discovery** → Interviste, analisi competitiva, definizione dei bisogni.
2. **Define** → Personas, scenari d'uso, requisiti funzionali UX.
3. **Design** → Wireframe (bassa fedeltà) → Prototipi interattivi (alta fedeltà).
4. **Evaluate** → Test di usabilità e iterazioni di miglioramento.
