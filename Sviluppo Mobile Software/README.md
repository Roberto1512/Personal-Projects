# Sviluppo Mobile Software — AsilApp

## Descrizione

Progetto relativo al corso di **Sviluppo Mobile Software**, dedicato alla progettazione e allo sviluppo dell'applicazione mobile **AsilApp**, realizzata dal team "The Verifiers".

La cartella contiene il codice sorgente (predisposto) e una ricca documentazione che copre l'intero ciclo di vita del prodotto: dalla specifica tecnica alla manualistica utente, fino al materiale di marketing.
